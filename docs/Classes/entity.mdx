---
title: Entity
---

The main entity object for BeAPI

:::warning

You should never have to construct this class yourself.

:::

## Constructor
```js
new Entity(client, entity);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| client | [Client](/docs/classes/client) |  ❌   | `null`  | Constructed [Client](/docs/classes/client) instance. |
| entity | [IEntity](https://docs.microsoft.com/en-us/minecraft/creator/scriptapi/mojang-minecraft/entity) |  ❌   | `null`  | Minecraft IEntity Object. |
