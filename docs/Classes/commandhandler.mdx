---
title: CommandHandler
---

A sub hub for creating and managing more customized commands.
:::warning

You should never have to construct this class yourself.

```js
import { client } from 'beapi-core';

const newHandler = client.commands.createHandler(...);
```

:::

## Constructor

```js
new CommandHandler(client, cm, prefix);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| client | [Client](/docs/classes/client) |  ❌   | `null`  | Constructed [Client](/docs/classes/client) instance. |
| cm | [CommandManager](/docs/classes/commandmanager) |  ❌   | `null`  | Constructed [CommandManager](/docs/classes/commandmanager) instance. |
| prefix | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `null`  | Prefix to use. |

## Methods

### changePrefix
Changes handlers prefix to something different.
```js
changePrefix(prefix);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| prefix | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `undefined`  | Prefix new handler should use. |

#### Returns
void
___

### getPrefix
Get handlers current prefix.
```js
getPrefix();
```

#### Returns
[string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)
___

### register
Register a command with the default handler.
```js
register(options, cb);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| options | [CommandOptions](/docs/typedefs/commandoptions) |  ❌   | `undefined`  | Options for command. |
| cb | [CommandCallback](/docs/typedefs/CommandCallback) |  ❌   | `undefined`  | Callback function called on usage. |

#### Returns
void
___

### unregister
Unregister a command on the default handler.
```js
unregister(name);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| name | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `undefined`  | Name of the command. |

#### Returns
void
___

### get
Gets a command on the default handler.
```js
get(name);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| name | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `undefined`  | Name of the command. |

#### Returns
[Command](/docs/typedefs/command) | [undefined](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/undefined)
___

### getAll
Gets all commands on the default handler.
```js
getAll();
```

[Map](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Map)<[string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String), [Command](/docs/typedefs/command)>
___
