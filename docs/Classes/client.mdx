title: Client

#### Extends [EventEmitter](/docs/classes/eventemitter)

The main hub for interacting with the higher level gametest API.

:::warning

You should never have to construct this class yourself.
Timers depend on its initialization before being exported.

```js
import { client } from 'beapi-core';

client.executeCommand('say Hello :)');
```

:::

## Constructor

```js
new Client();
```

### verifyIEvent
Ensures given event exists in gametests events.

```js
verifyIEvent(name);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| name | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | ❌ | `undefined` | The name of the event. |

:::note
The name "custom" will return `true` because this is telling us the event is not provided by Minecraft.
:::

#### Returns
[boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean)
___

### executeCommand
Execute a command as system admin.

```js
executeCommand(cmd, dimension, debug);
```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| cmd | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `undefined`  | Minecraft command to execute. |
| dimension | [Dimension?](/docs/typedefs/Dimension) |  ✔️   | `overworld`  | Dimension to execute given command in. |
| debug | [boolean?](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) |  ✔️   | `false`  | Pipe logs to debug console. |

#### Returns
[ServerCommandResponse](/docs/typedefs/servercommandresponse)
