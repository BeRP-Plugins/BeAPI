---
title: World
---

The main world object for BeAPI

:::warning

You should never have to construct this class yourself.

:::

## Methods

### sendMessage
Sends a message to every player.

```js
sendMessage(message);
```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| message | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `undefined`  | Message to send. |

#### Returns
void
___

### getEntitiesFromLocation
Gets all the entities in a specific location.

```js
getEntitiesFromLocation(dimension, location);
```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| dimension | [Dimension](/docs/typedefs/dimension) |  ❌   | `undefined`  | Dimension to locate in. |
| location | [Location](/docs/typedefs/location) |  ❌   | `undefined`  | Location to check. |

#### Returns
[Entity[]](/docs/classes/entity)
___

### spawnEntity
Spawns an entity.

```js

```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| entity | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `undefined`  | Entity id. |
| location | [Location](/docs/typedefs/location) |  ❌   | `undefined`  | Location to summon. |
| name | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `undefined`  | Name of the entity. |

#### Returns
[Entity](/docs/classes/entity)
