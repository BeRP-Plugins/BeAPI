Exported method to create a BeAPI interval.

:::caution

Intervals and timeouts need a custom implementation due to how gametest works. You must use them instead of the default JS implementations.

:::

## Method

```js
setInterval(callback, tick);
```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| callback | <Link to="https://microsoft.github.io/PowerBI-JavaScript/interfaces/_node_modules_typedoc_node_modules_typescript_lib_lib_es5_d_.callablefunction.html">CallableFunction</Link> | ❌ | `null` | Callback function. |
| tick | <Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number">number</Link> | ❌ | `null` | Tick speed. |

## Returns
<Link to="/typedefs/interval">Interval</Link>

## Examples

```js
import {
  clearInterval,
  setInterval,
} from 'beapi-core';

// Create a variable with 0
let runtime = 0
// Create an interval the runs every 10 ticks
const interval = setInterval(() => {
  // If variable is more than or equal to 10 kill the interval
  if (runtime >= 10) return clearInterval(interval)
  // Log the current interval number and increment it by 1
  console.warn(`Interval #${runtime++}`)
}, 10)
```
