> **<Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/class">class</Link>**

Exported class that enables mounting of an ingame entity and controlling said entity.

## Constructor

```js
new Entity(entity);
```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| entity | <Link to="https://docs.microsoft.com/en-us/minecraft/creator/scriptapi/mojang-minecraft/entity">MCEntity</Link> | ❌ | `null` | Mc entity object. |
