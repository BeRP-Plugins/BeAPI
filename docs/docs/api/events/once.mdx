> **<Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function">function</Link>**

Exported method that enables registering a listener for when an event is envoked. It will only run once however, after the event is envoked once it removes itself.

## Method

```js
once(event, callback);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| event | <Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String">string</Link> | ❌ | `undefined` | Event name. |
| callback | <Link to="https://microsoft.github.io/PowerBI-JavaScript/interfaces/_node_modules_typedoc_node_modules_typescript_lib_lib_es5_d_.callablefunction.html">CallableFunction</Link> | ❌ | `undefined` | Callback. |

:::note

See <Link to="/docs/typedefs/beapievents">BeAPI Events</Link> for a list of all events.

:::

## Returns
void

## Examples
```js
import { events } from 'beapi-core';

// Log first block broken
events.once('BlockDestroyed', console.warn);

// Log the name of first block placed
events.once('BlockPlaced', (event) => {
  console.warn(event.block.id);
});
```
