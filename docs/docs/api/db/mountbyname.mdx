Exported method to get persistent database by its name, otherwise create it.

## Method

```js
mountByName(name);
```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| name | <Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String">string</Link> | ‚ùå | `null` | Database name. |

## Returns
<Link to="/docs/api/db/Database">Database</Link>

## Examples

```js
import { db } from 'beapi-core';

// Create new database
const database = db.mountById('foobar')
// Add entry
database.addEntry({
  name: 'foo',
  value: 'bar',
})
```
