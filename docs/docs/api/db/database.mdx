Exported class that allows creation a new database or get an persistent database.

:::caution

Even though you can use this class to create and get a database, it is not recommended.

We recommend using the methods: <Link to="/docs/api/db/mountbyid">mountById</Link> or <Link to="/docs/api/db/mountbyname">mountByName</Link>
instead. these methods will get the database if it already exists otherwise create it.

:::

## Constructor

```js
new Database(options, mount);
```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| options | <Link to="/docs/typedefs/dboptions">DbOptions</Link> | ❌ | `null` | Database options. |
| mount | <Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean">boolean</Link> | ✔️ | `null` | Whether to just mount or create. |
