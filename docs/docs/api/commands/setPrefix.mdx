Exported method that sets BeAPIs default command handler prefix.

## Method

```js
setPrefix(prefix);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| prefix | <Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String">string</Link> | ‚ùå | `undefined` | New prefix. |

## Returns

void

## Examples

```js
import { commands } from 'beapi-core';

// Set BeAPIs command prefix to "+"
commands.setPrefix('+');
```
