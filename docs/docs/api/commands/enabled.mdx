> **<Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean">boolean</Link>**

Exported property that indicates whether BeAPIs default command manager is enabled or disabled. You can also use this to disable it or enabled it.

## Examples

```js
import { commands } from 'beapi-core';

if (commands.enabled) console.warn('BeAPI Command Manager is Enabled');
else console.warn('BeAPI Command Manager is Disabled');
```

```js
import { commands } from 'beapi-core';

// If commands are enabled, disable them
if (commands.enabled) commands.enabled = false;
```
