> **<Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function">function</Link>**

Exported method that gets BeAPIs default command handlers prefix.

## Method

```js
getPrefix();
```

## Returns

<Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String">string</Link>

## Examples

```js
import { commands } from 'beapi-core';

// Get command prefix
const prefix = commands.getPrefix();
// Log current prefix
console.warn('BeAPIs current prefix:', prefix);
```
