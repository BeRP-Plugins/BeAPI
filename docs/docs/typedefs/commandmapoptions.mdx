Command map options object, contains command data.

## Properties

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| options | <Link to="/docs/typedefs/commandoptions">CommandOptions</Link> | ❌ | `undefined` | Command option data. |
| showInList | <Link to="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean">boolean</Link> | ❌ | `undefined` | Command shown in help list. |

## Methods

### execute
Envoke logic for when a player invokes a said command.

```js
execute(data);
```

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| data | <Link to="/docs/typedefs/commandresponse">CommandResponse</Link> | ❌ | `undefined` | Logic to be called when command is executed in game. |

void
