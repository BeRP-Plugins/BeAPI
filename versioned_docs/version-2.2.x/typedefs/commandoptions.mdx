## Type
- [object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object)

## Properties

| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| name | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `null`  | Name of command. |

| hidden | [boolean?](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) |  ✔️   | `false`  | Show command in help list? |
