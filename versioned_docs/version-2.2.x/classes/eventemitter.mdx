---
title: EventEmitter
---

A purejs provided solution to polyfill [NodeJS EventEmitter](https://nodejs.org/api/events.html).

## Contructor
```js
new EventEmitter(max);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| max | [number?](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) |  ✔️   | `50`  | Max number of allowed listeners before a mem leak warning is emitted. |

## Methods

### addListener
Add a new listener to be synchronously called on an event.
```js
addListener(event, listener);
```
| PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION |
| :-------: | :--: | :------: | :-----: | :---------: |
| event | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) |  ❌   | `undefined`  | Event name to add listener to. |
| listener | [CallableFunction](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Functions) |  ❌   | `undefined`  | Function to execute. |

#### Returns
void
___
